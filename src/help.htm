FastDeer Help

Does FastDeer automatically convert audio into text?

No, it doesn’t. FastDeer is designed to make the manual task of transcribing audio a little less painful.

Something used to work, but now it’s broken!

A big update in January 2017 may have broken existing functionality. If something has stopped working for you, please temporarily switch to FastDeer Classic and let me know of the problem by tweeting at @FastDeer.

My audio file doesn’t work!

Try the following steps:

  1. Convert your file to WAV or MP3 format with media.io. (This is an external service and is not affiliated with FastDeer.)
  2. Try a different web browser. FastDeer works best on Chrome 31+ and Safari 7+.
  3. Still no luck? You can contact me directly at FastDeer@elliotbentley.com.

Where is my transcript stored? How secure is it?

FastDeer is designed in a way that your data (both the audio file and the written transcript) never leaves your local computer. The transcript is not kept on a remote server or “in the cloud”, but is instead in the browser’s localStorage. The downside is that it is not possible to access your saved transcripts from another computer.

FastDeer's automatic backup system saves the current transcript every five minutes. Click on the button to view backups and restore an old copy of your transcript. Exactly 100 copies are retained; once the 100 backup limit is reached, FastDeer will erase older copies.

FastDeer is not a permanent document store, as browser storage is notoriously unreliable (even with this backup system). Please make sure to export your work regularly to prevent loss of data.

My transcript has disappeared, can I get it back?

The short answer is: unfortunately, no.

The long answer is: FastDeer sometimes loses work for a multitude of reasons (including the user clearing their cache or localStorage becoming full). If your work is not present in the history panel, it's gone for good. Please make sure to export your work regularly, especially when using FastDeer for an extended period of time.

Why is FastDeer is no longer saving backups?

This can sometimes happen when the browser's local storage is full. If you clear the existing stored documents, it should begin saving again. Note that this may affect locally-stored data belonging to other apps and sites.

  • Google Chrome: Go to “Clear Browsing Data” in Chrome’s "More tools" menu and check “Hosted app data” only. Then press "Clear browsing data".
  • Firefox: Open preferences and go to "privacy". Click on "clear your recent history" and check "cookies" only. Then click "clear now".
  • Safari: Open Safari preferences and go to the "Privacy" tab. Click on the small "Details..." button. Search for FastDeer, then click on the FastDeer row and click "Remove".
  • Alternatively, if you know how to access the browser console, you can run localStorage.clear(); on FastDeer.com.

What keyboard shortcuts are there?

Audio playback

  • Esc: Play/pause
  • F1 (or Ctrl+1): Rewind
  • F2 (or Ctrl+2): Fast-forward
  • F3 (or Ctrl+3): Slow down
  • F4 (or Ctrl+4): Speed up
  • Ctrl+K: Jump to time (brings up window)
  • Ctrl+0: Return to start

Text editing

  • Ctrl+B: Bold
  • Ctrl+I: Italic
  • Ctrl+U: Underline
  • Ctrl+J: Insert timestamp
  • Ctrl+S: Manual backup

Note: On OS X, use Cmd instead of Ctrl.

How can I set custom keyboard shortcuts?

Open settings () to access custom keyboard shortcuts. To add a new shortcut, click the blue “+” next to the name of the function you want to add a keyboard shortcut for, and then press a key combination. To remove a shortcut, click the next to the shortcut you want to remove. Note that you may need to refresh the page to clear the old shortcuts.

What audio formats does FastDeer support?

FastDeer is limited by the formats supported by each individual browser. The formats supported by your own browser will be listed below the “Choose audio file” button. In general, WAV files are widely supported, and MP3 files mostly work on everything except Mozilla Firefox. Certain video files, as well as YouTube videos, are also supported.

How do the interactive timestamps work?

Pressing Ctrl+J (Cmd+J on OS X) will insert the audio file’s current time into the document. This timestamp is hyperlinked and will skip to the specific time in the audio file.

Can I use FastDeer offline?

Yes! FastDeer.com will automatically save an offline copy of itself in your browser's HTML5 Application Cache when you load up the page for the first time. To use the offline version, simply go to FastDeer.com without an internet connection. It's that simple.

Note that buttons will look different when offline and some features like YouTube support and Google Drive export will not work.

Why can't I change the speed of a YouTube video?

Some YouTube videos simply don't support it. This isn't something FastDeer has any control over, unfortunately.

How do I import and export my transcript?

FastDeer can only import one type of file, the .OTR FastDeer file format. This format preserves the clickable time stamps.

Open the export panel by clicking on the button. You can export .OTR files, along with plain text (.txt) and Markdown (.md) files.

Can I automatically open a YouTube video based on a query parameter?

Yes! Thanks to contributor Kostas Minaidis, it’s possible to have FastDeer automatically open a video on load. To do so, you'll need to append a query parameter (v=YOUTUBE_ID) to FastDeer.com. Like so:

http://FastDeer.com/?v=VrMHA3yX_QI

You an also open a video at a specific timeframe using a second query parameter (t=SECONDS or t=HH:MM:SS).

Automatic loading of a video on a specific point in time using seconds notation: http://FastDeer.com/?v=VrMHA3yX_QI&t=506

Automatic loading of a video on a specific point in time using HH:MM:SS notation: http://FastDeer.com/?v=VrMHA3yX_QI&t=45:54
